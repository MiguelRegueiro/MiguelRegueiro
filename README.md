# Miguel Pérez Regueiro

Computer Engineering student in Spain building software for Linux, the terminal, and everyday workflows.

I build tools to remove friction from how I use my computer. I want them to be fast, clear, and good to use.

My main project is [elio](https://github.com/elio-fm/elio) — a terminal file manager in Rust with rich previews, inline images, theming, and distribution through AUR, COPR, Homebrew, and crates.io.

Other projects:

- [runin](https://github.com/MiguelRegueiro/runin) — launcher I use daily to jump into projects and config folders
- [ssh-watchdog](https://github.com/MiguelRegueiro/ssh-watchdog) — GNOME Shell extension for monitoring SSH sessions
- [rustnake](https://github.com/MiguelRegueiro/rustnake) — terminal Snake game built in Rust
